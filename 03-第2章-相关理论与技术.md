第 2 章 相关理论知识和技术介绍

2.1 Spring Boot 框架介绍

Spring Boot 提供自动配置与起步依赖机制，可显著降低 Spring 应用的配置复杂度。其内嵌服务器与统一配置方式适合构建 RESTful 服务，为招生管理系统的接口开发提供稳定支撑。本系统基于 Spring Boot 组织业务逻辑与接口调用，结合统一异常处理与拦截机制提升系统稳定性。

2.2 Vue.js 前端框架

Vue.js 采用组件化与响应式数据绑定机制，适合构建管理类系统的交互界面。其路由与组件复用能力能够降低页面开发成本，便于实现公告管理、报名管理与通知管理等模块的界面交互。本系统在管理端使用 Vue.js 构建操作界面，提高页面响应效率与可维护性。

2.3 MyBatis‑Plus 数据访问框架

MyBatis‑Plus 在 MyBatis 基础上提供通用 CRUD、分页与条件构造能力，减少重复代码并提高数据访问层开发效率。其与 Spring Boot 集成便捷，适用于招生公告、专业信息与报名数据等多表场景的查询与维护。

2.4 MySQL 数据库

MySQL 具备成熟的事务机制与索引优化能力，适合管理类系统的高频数据读写。招生管理系统需要存储公告、专业、报名与录取通知等核心数据，通过合理的索引与事务控制可保障数据一致性与查询效率。

2.5 Java 语言

Java 具备跨平台、生态成熟与工程化能力强等特点，是企业级应用开发的主流语言。基于 Java 的后端系统能够通过分层结构实现模块解耦，满足业务扩展与维护需求，为招生管理系统的长期运行提供基础保障。
