Abstract

Enrollment management in educational institutions involves information publishing, major setup, application review, and admission notice delivery. Traditional offline processes suffer from delayed information flow, fragmented procedures, and difficult statistical analysis. This thesis designs and implements an enrollment management system based on Spring Boot. The system adopts a B/S architecture with front‑end and back‑end separation. The back end is built with Spring Boot and MyBatis‑Plus, while the management interface is implemented with Vue.js, and MySQL is used for centralized data storage. The system serves administrators and students, providing enrollment announcement management, major information management, application processing, admission notices, student information management, system configuration, and file upload services, with role‑based access control and auditability.

Functional verification shows that the system can reliably support the enrollment workflow and improve management efficiency for educational institutions.

Keywords: enrollment management system; Spring Boot; MyBatis‑Plus; MySQL; access control
